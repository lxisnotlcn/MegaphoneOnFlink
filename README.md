# MegaphoneOnFlink

## 项目介绍

基于Flink1.12.0实现megaphone，由于Flink系统与论文本身使用的Timely dataflow系统特性的差异，本系统并没有完全按论文的控制流程进行复现，请见：

[论文复现：基于Flink实现Megaphone | lxisnotlcn's blog](http://lxisnotlcn.love/2023/11/28/MegaphoneOnFlink/)
